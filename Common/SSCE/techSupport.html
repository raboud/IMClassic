Wintertree Software Technical Support

Technical support is available via Wintertree Software's web site and e-mail to help you integrate Sentry Spelling Checker Engine with your application. Before contacting us for technical support, please do the following:

  • Read the Sentry Programmer's Guide carefully.

  • Read the readme.htm file distributed with the Sentry SDK.
  • Visit the Sentry Support Page for answers to frequently asked support questions, solutions to common problems, background information, and tips.

We prefer to answer technical support questions via e-mail. Please provide your Sentry serial number when contacting us for support.
